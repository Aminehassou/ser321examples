You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns the result of multiplication between two integers. Example: "/multiply?num1=2&num2=3"
  • /github -- returns the full names of a GitHub user's repositories, id of each repository, and names of the repository owners. Example: "/github?query=users/amehlhase316/repos"
  • /base64 -- encodes or decodes the given string and returns the result. Takes 2 parameters, the message to encode/decode and the type of operation (encode or decode).
    Example 1: "/base64?message=hello&operation=encode"
    Example 2: "/base64?message=aGVsbG8=&operation=decode"
  • /anagram -- checks if 2 words are anagrams of eachother and returns the result. Example: "/anagram?string1=rad&string2=dar"
